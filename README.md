Project Description: 
-
A website application designed to help people in need access clothes to get it free. Upload or post an item with clothes he doesn’t need by the owner user with the information about the item. Get a request from the user who would like to get the item, after the item can be delivered to the user.

3 models: 
-
User register.
Clothing Item Model: post clothes that the user wants to donate:
-	Store all the information about clothes items.
-	Allow the user to add an item.
-	Edite or delete the item.
-	View available donation.
Request Model: get a request from the other user for a specific item.
-	Communication between both users to get the items.
-	Get request.
-	Send message to the owner of the item.

Getting Started: -
-
Sign-up And Sign-in, Add clothes by the user, Other users check the available items, Get a request to get the item, The owner of the item receives the request and response, Have Full CRUD (CREATE, READ, UPDATE, DELETE).
